Motivation

This project was originally for my wife, Erin, who recently expressed an interest in web design. Finding no interactive HTML tutorials (though plenty of interactive editors), I based this roughly on the W3Schools tutorial, and heavily inspired (obviously) by codecademy. Much of the initial coding was actually done around 3 am on a sleepless night when I finally figured it had to be done.

Contributing

This project is hosted at github, and I encourage you to edit, add content, etc. if you feel so inclined. The easiest way to do so is to just look at the first lesson for an example of how to add additional lessons. Also, as it is glaringly obvious, I don't do a lot of markup or web design and would welcome any design considerations.
